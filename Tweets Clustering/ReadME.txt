The src folder contains the following files:
Kmeans.java
TweetData.java
clusterTweets.java

Steps to compile and run:
To compile the files:
Traverse to the path where the source files are present . Open command line window  there.
javac <filename>(for all the three files)
The above command will create the .class files.

To run the file
java Kmeans <numberOfClusters><initialSeedsFile><TweetsDataFile> <Outputfilename>
